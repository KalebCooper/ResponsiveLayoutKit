# Security Policy

## Supported versions

ResponsiveLayoutKit is a pre-1.0 library. Security fixes land on the latest minor release only. Upgrade to the current minor to receive them.

| Version | Supported |
|---|---|
| 0.2.x | :white_check_mark: |
| < 0.2 | :x: |

## Reporting a vulnerability

Report vulnerabilities privately through GitHub security advisories, not public issues or pull requests.

1. Open the repository's **Security** tab.
2. Choose **Report a vulnerability** (or go to https://github.com/KalebCooper/ResponsiveLayoutKit/security/advisories/new).
3. Describe the issue, the affected version, and a minimal reproduction if you have one.

Expect a best-effort acknowledgement within a few days. Fixes follow coordinated disclosure: please keep the report private until a patched release is available. Valid reports are credited in the release notes unless you ask otherwise.
